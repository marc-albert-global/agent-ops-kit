---
name: activation-definition-change
description: The activation event definition changed in Q1, breaking funnel comparability
type: reference
created: 2026-03-01
---

The "activation" milestone was redefined in Q1 from "first login" to "first
value action completed". Funnel conversion rates that span the Q1 boundary are
not comparable without restating the pre-Q1 numbers under the new definition.
Flag this whenever a Signup→Activation comparison crosses Q1.
